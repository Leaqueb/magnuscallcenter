## Como iniciar?

Instale o MagnusCallcenter no seu servidor com CENTOS 7

### Prerequisitos

Linux Centos 7


### Instalação
```
cd /usr/src/
yum -y install wget
wget https://raw.githubusercontent.com/magnussolution/magnuscallcenter/master/script/install.sh
chmod +x install.sh
./install.sh

```
